Escenario

En esta actividad realizaras una aplicación Web (un conjunto de uno o más archivos o páginas Web) y aplicarás los elementos mostrados en las secciones “Estilos en páginas Web” y “Tareas básicas de HTML”.

Resumen del módulo y actividad

La etiqueta <p> se usa para agrupar segmentos de texto en párrafos

El tag <em>, que proviene de emphasized, sirve para hacer énfasis en un texto

El uso del tag <strong> sirve para resaltar, a manera de negritas, un texto

Tanto el tag <em> como <strong> aún son vigentes en HTML5 pero es posible agregar los mismos efectos usando CSS3

Los caracteres especiales como © deben ser representados con sus respectiva versión en código HTML que sigue el formato &"codigo" donde "codigo" es una cadena de caracteres que representa el caracter especial como lo es &copy para insertar el símbolo ©.

Una lista de códigos HTML para caracteres especiales puede ser consultada en esta dirección

También podemos usar tipografías personalizadas mediante la regla @font-face. Para mayor información sobre esta característica, podemos visitar la página de w3school en la sección correspondiente a CSS3 @font-face Rule
