Escenario

En esta actividad realizaras una aplicación Web (un conjunto de uno o más archivos o páginas Web) y aplicarás los elementos mostrados en el módulo “Incorporando Imágenes”.

Resumen del módulo y actividad

El uso de imágenes en una página web nos permite darle más dinamísmo que al usar únicamente texto. Dichas imágenes pueden ser utilizadas para diferentes propósitos como puede ser un fondo para nuestra página, crear vínculos a otras páginas, hacer un mapa de imágen o simplemente acompañar nuestro texto. La inserción de imágenes se realiza principalmente a través de la etiqueta <img> especificando en su atributo src la dirección a la imágen

Imágenes flotantes

Las imágenes flotantes son la forma más básica de insertar imágenes en nuestra página de internet ya que podemos combinarla con texto sin mayor complicación. Las imágenes se alinearán al igual que la alineación del texto que las contiene pero podemos alterar este comportamiento mediante CSS usando el atributo float en la imagen deseada.

También podemos agregar un margen a la imagen para que exista una separación con respecto al texto. Este margen lo podemos colocar alrededor de la imagen usando el atributo margin o alguna de sus variantes como lo son margin-left, margin-right, margin-top o margin-bottom para agregar un margen sólo en el lado especificado.

La imágen de la derecha utiliza un margen de 10px en todos sus lados mientras que la de la izquierda tiene un margen de 100 pixeles en su lado derecho


Imágenes de fondo

Las imágenes de fondo permiten sustituir el color de fondo de la página por una imágen de nuestra elección logrando con esto poder personalizarla a nuestro antojo

La imágen de fondo se puede definir mediante CSS utilizando la propiedad background-image dentro de los atributos de body utilizando url(""); con la ruta de la imágen. También mediante CSS, podemos definir si la imágen se repetirá mediante background-repeat o especificar su posición a través del uso de la propiedad background-position y con background-attachment puedes definir si la imágen se quedará estática o se desplazará junto con la página

Esta página contiene una imagen de fondo pero puedes apreciarla de mejor manera en esta página

Imágenes como enlace

Las imágenes también se pueden combinar con los hipervínculos para generar enlaces más llamativos que los que el uso exclusivo de texto podría brindarnos Para hacer esto, solo se cambia el texto a desplegar en el enlace por la liga a la imágen deseada usando la etiqueta <img>

Mapas de imágen

Un mapa de imágen nos permite incrustar varios enlaces utilizando una misma imágen y agregando formas que no son visibles pero que el usuario puede hacer clic en ellas logrando con esos un mayor grado de interactividad.

Es importante que, al realizar una página web, probemos nuestros diseños en la mayor cantidad de navegadores posibles o al menos en los principales que aparecen a continuación. Si deseas descargar alguno puedes hacer clic en su logotipo para ir a la página de descarga

Reglas horizontales como imágen

Las reglas horizontales se crean con la etiqueta <hr> y sirven para dibujar una linea divisoria horizontal utilizada para separar contenido

Al igual que otros elementos HTML, las reglas horizontales también pueden ser personalizadas mediante el uso de CSS
