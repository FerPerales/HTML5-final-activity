Escenario

En esta actividad realizaras una aplicación Web (un conjunto de uno o más archivos o páginas Web) y aplicarás los elementos mostrados en el módulo “Utilizando listas”.

Resumen del módulo y actividad

Las listas son una herramienta para representar información que tiene alguna clase de relación.

En HTML5 existen dos tipos de listas principalmente:

  • Listas desornedadas: representadas con la etiqueta <ul>
  • Listas ordenadas: representadas con la etiqueta <ol>

En cualquiera de las los listas, los items son representados con la etiqueta <li>

El ejemplo anterior emplea una lista desordenada ya que no fue necesario catalogar o dar más prioridad a algún elemento

También podemos cambiar el tipo de viñeta que se muestra por defecto y ponerle una imagen personalizada a través de CSS haciendo uso de la propiedad list-style-image:url() y dentro de url, la dirección de la imagen deseada

Puntos a tratar sobre listas:

  • Listas ordenadas
  • Listas con imágen como viñeta
  • Listas ordenadas
  • Listas como barra de navegación

Las listas ordenadas suelen mostrar números consecutivos para denotar un orden

La forma de desplegar las listas varía según el navegador y debemos tomar en cuenta esto para hacer un sitio que sea o intente ser 100% compatible con la mayoría de los navegadores principales que, según el artículo de Wikipedia respecto al porcentaje de uso de los navegadores, corresponde a lo siguiente:

  1. Internet Explorer - 34.2%
  2. Firefox - 23.6%
  3. Chrome - 20.6%
  4. Safari - 11.2%
  5. Opera - 5.0%

Lista de definiciones

Una lista de definiciones es representada con la etiqueta <dl> y nos permite hacer listas que representan una asociación correspondiente de dos elementos llamados término y descripción que son representados con las etiquetas <dt> y <dd> respectivamente

Puede usarse para hacer una lista con palabras y definiciones o preguntas y respuestas tal como se puede observar en la sección de preguntas de esta página

Otro de los usos para las listas de definiciones es para sustituir a la etiqueta <dialog> que aún no está considerada como parte del estándar

Barra de navegación

Una barra de navegación es una especia de menú que nos permite desplazarnos a través de diverso contenido web ya sea en nuestra página o enlazando otras páginas.

Usando una lista ordenada y CSS, podelos crear una barra de navegación, para ahorrar algo de tiempo, podemos incluir algunas de las barras de navegación de libre uso ya existentes en la web o puedes ver alguno de los tutoriales para crearla
