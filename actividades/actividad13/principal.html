Escenario

En esta actividad realizaras una aplicación Web (un conjunto de uno o más archivos o páginas Web) y aplicarás los elementos mostrados en el módulo “Estructurando Tablas”.

Resumen del módulo y actividad

Las tablas originalmente se utilizaban para la maquetación de un sitio web usando la tabla como elemento principal y en cada una de las celdas se colocaban los componentes de nuestra página.

Hoy en día el uso más común es la representación de la información a manera de cuadrícula como se haría en una hoja de cálculo.

Para crear una tabla, la etiqueta a utilizar es <table> e internamente se usan las etiquetas <tr> para agrupar a manera de columnas y <td> para representar cada una de las celdas de la tabla.

Tabla con encabezado, cuerpo y pie de tabla

Adicionalmente, se pueden utilizar otras etiquetas para definir las partes de una tabla que son encabezado, cuerpo y pie de tabla.

La etiqueta <thead> nos sirve para definir la fila que actuaría como encabezado en nuestra tabla conteniendo los títulos de la misma. Para definir las celdas que contendrán estos títulos, se utiliza la etiqueta <th>

El cuerpo de la tabla es la parte que contiene los datos a tabular y se denota con el uso de la etiqueta <tbody>

Por último, podemos utilizar la etiqueta <tfoot> para indicar el pie de tabla que suele contener información como totales o algún otro dato que ayude a resumir los valores de la tabla

Tabla con título (Caption-Details-Summary) e imagen de fondo

Al usar tablas, es conveniente saber que podemos utilizar algunas etiquetas adicionales para agregar información adicional sobre la tabla que puede o no ser mostrada por el navegador pero que facilitará la comprención de la tabla.

La etiqueta <caption> es la primera que se usa dentro de la etiqueta <table> y sirve para poner una descripción de la tabla. Usando CSS y la propiedad caption-side, podemos indicar la posición con respecto a la tabla

Dentro de la etiqueta <caption>, podemos agregar la etiqueta <details> para añadir información adicional sobre la tabla

Por último, dentro de <details>, podemos usar la etiqueta <summary> y mostrarlo u ocultarlo mediante la propiedad open="open"

También podemos utilizar alguna imágen de fondo para usarla en toda la tabla usando la etiqueta <img>
