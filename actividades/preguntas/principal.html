Módulo 5: Incorporando imágenes

1. ¿Cuáles son las principales diferencias entre los formatos de imágenes GIF, JPEG y PNG?

GIF (Graphics Interchange Format) Una imagen GIF consiste de 256 colores o menos y es utilizado para gráficos que tiene áreas de colores limitados, como logotipos e ilustraciones. Las imágenes GIF también tienen áreas transparentes y son utilizados para dar la apariencia de gráficos no rectangulares. Los archivos GIF tienen una extensión .gif. Reducir el tamaño de una imagen GIF implica eliminar colores. El formato GIF soporta animación a través de una arquitectura de intercambio de páginas. En una animación GIF una serie de imágenes se muestran rápidamente para dar la apariencia de movimiento. Los GIFs animados se utilizan frecuentemente para mostrar banners con anuncios en la web.

JPEG (Joint Photographic Experts Group) Una imagen fotográfica se representa mejor en un formato JPEG. Una imagen JPEG puede contener miles de colores, necesarias para mostrar rangos de colores que aparecen en una fotografía. A diferencia de las imágenes GIF, las imágenes JPEG no pueden mostrar áreas transparentes. Para disminuir el tamaño de un archivo JPEG se debe reducir la calidad de la imagen. El problema es que cuando se reduce la calidad de la imagen esta se vuelve menos legible.

PNG (Portable Network Graphics) Lo mejor de los dos mundos es el formato PNG. Este formato ofrece un amplio rango de colores como JPEG y ofrece también transparencia superior a la de imagen GIF. PNG ofrece diversos formatos de colores como 8 bits, 16 bits y 24 bits; lo cual nos permite optimizar las imágenes de forma apropiada.

2. ¿En qué situaciones es recomendable utilizar cada uno de formatos de imágenes?

GIF para imagenes con pocos colores y animaciones, JPG imágenes en general y PNG para imágenes con transparencia

3. ¿Cuáles son los atributos principales del TAG img?

4. ¿Cuál es la forma de mostrar imágenes como enlaces?

Insertando una imagen dentro de la etiqueta <a>

5. ¿De qué manera se puede alinear una imagen junto con un texto?

Usando la propiedad float

Módulo 9: Mejorando HTML con JavaScript

1. ¿Qué es JavaScript?

Lenguaje de programación interpretado. Es utilizado especialmente en páginas web embebido en el código HTML o similares

2. ¿Qué significa que JavaScript sea un lenguaje interpretado?

No es necesaria la compilación la que el contenido se interpreta por el navegador

3. ¿Qué es DOM?

Document Object Model ("Modelo de Objetos del Documento"), es una interfaz de programación de aplicaciones para acceder, añadir y cambiar dinámicamente contenido estructurado en documentos con lenguajes como JavaScript, se conoce también como "modelo básico"

4. ¿Por qué se dice que JavaScript utiliza una notación con puntos?

Porque para identificar un elemento JavaScript se utilizan puntos para separar los elementos de la notación

5. ¿Cuáles son algunas de las posibles formas de incorporar código JavaScript a una página Web?

Se puede incorporar código durante la carga de la página, después de la carga de la página yse puede incorporar sobre demanda

Módulo 10: Agregando audio y video

1. ¿Qué es un Plug-In?

Es un pequeño programa que le agrega funcionalidad a una página web

2. ¿Cuáles son los TAGs que permiten incluir contenido para que sea procesado por un Plug-In?

<object> y <embed>

3. ¿Cuál es el TAG que nos permite repoducir audio en una página Web sin necesidad de utilizar Plug-Ins?

<audio>

4. ¿Cuáles son algunos de los formatos de video disponibles para ser mostrados en una página Web?

FLV, MP4, OGG, WEBM y WMV

5. ¿Cuáles son algunos de los formatos de audio disponibles actualmente para poder ser incorporados en páginas Web?

MP3, WAV, OGG
